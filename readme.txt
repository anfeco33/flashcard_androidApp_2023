HƯỚNG DẪN CHẠY TRÊN ANDROID STUDIO

BƯỚC 1: Mở Android Studio và chọn clone link git sau về một thư mục trống "https://gitlab.duthu.net/S52100965/flashcardlearningenglish.git"

BƯỚC 2: Mở thư mục 'src' trong thư mục vừa clone về và chờ hệ thống build (Nếu gặp lỗi về file local.properties trong Gradle Scripts (thiếu file local.properties hoặc sai đường dẫn)
	1/ Thiếu file: Tạo file tên local.properties và thêm dòng lệnh 'sdk.dir=C\:\\Users\\[Tên thư mục trên máy người dùng]\\AppData\\Local\\Android\\Sdk')
	2/ Sai đường dẫn: Về lỗi này máy có thể tự động sửa lại, nếu vẫn lỗi bạn có thể trực tiếp sửa lại đường dẫn này trong file local.properties với 'sdk.dir=C\:\\Users\\[Tên thư mục trên máy người dùng]\\AppData\\Local\\Android\\Sdk')

BƯỚC 3: Chọn thiết bị có hệ điều hành android 5. Nhấn run để chạy chương trình ứng dụng, thiết bị mới tải sẽ có thể phải đợi lâu hơn.
